[Directories]

GENIAcorpus_event
	It contains 1,000 XML files.
	Each file contains one Medline abstract.
	Term annotation is inserted in sentences.
	Event annotation is attached to each sentence.

GENIAtypes
	It contains the DTD and CSS files for the XML files constituting the GENIA event corpus.
	All the XML files in the corpus are validated against the DTD.
	The CSS well works with the Opera web browser.

GENIAontologies
	It contains two GENIA ontologies encoded in OWL.
	The GENIAterm40.owl defines the term classes on which the GENIA term annotation is based.
	The GENIAevent.owl defines the event classes on which the GENIA event annotation is based.
